# Efficio - Task Manager

Efficio is a practice project created following the TaskBuddy project. It is a powerful task management app built using ReactJS. It helps users manage daily tasks with ease â€” add, update, delete, track progress, and clear all tasks with just one click!

---

## Features


* **Add Tasks**: Users can add new tasks to the list.
* **Update Tasks**: Users can edit the details of existing tasks.
* **Delete Tasks**: Users can remove tasks from the list.
* **Clear All Tasks**: Users can clear all tasks with a single click.
* **Progress Tracker**: Visual representation of the task completion status.
